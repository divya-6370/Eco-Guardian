🌍 Plastic Usage Tracker

A web application that helps users track daily plastic usage, calculate the estimated CO₂ emissions, and visualize their environmental impact through interactive charts. The goal of this tool is to promote sustainable habits by making plastic consumption easy to monitor and understand.

📌 Features

📝 Daily Plastic Logging
Users can record the amount of plastic they use each day.

🌱 CO₂ Emission Calculation
Automatically estimates the carbon footprint generated from plastic usage.

📊 Interactive Data Visualization
Displays plastic usage trends using dynamic charts for better insights.

📁 CSV Export
Allows users to export their data for analysis or record keeping.

💡 Eco-Friendly Tips
Provides suggestions to help users reduce plastic consumption.

⏰ Usage Reminders
Alerts users when plastic usage becomes unusually high.
